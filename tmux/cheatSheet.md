# tmux チートシート

**Prefix**: `Ctrl + t`

---

## セッション

| キー | 説明 |
|------|------|
| `tmux new -s <name>` | セッション作成 |
| `tmux ls` | セッション一覧 |
| `tmux attach -t <name>` | セッションにアタッチ |
| `prefix + d` | セッションをデタッチ |
| `prefix + $` | セッション名変更 |
| `prefix + s` | セッション一覧・切替 |

---

## ウィンドウ

| キー | 説明 |
|------|------|
| `prefix + c` | 新規ウィンドウ作成（名前を指定） |
| `prefix + w` | ウィンドウ一覧表示 |
| `prefix + &` | ウィンドウを閉じる |
| `prefix + ,` | ウィンドウ名変更 |
| `prefix + n` | 次のウィンドウ |
| `prefix + p` | 前のウィンドウ |
| `prefix + <番号>` | 番号でウィンドウ移動 |
| `prefix + Ctrl-h` | 前のウィンドウ |
| `prefix + Ctrl-l` | 次のウィンドウ |

---

## ペイン

| キー | 説明 |
|------|------|
| `prefix + \|` | 縦分割 |
| `prefix + -` | 横分割 |
| `prefix + h/j/k/l` | ペイン移動（左/下/上/右） |
| `prefix + Ctrl-g` | ペイン番号表示して選択 |
| `prefix + H/J/K/L` | ペインリサイズ（5単位） |
| `prefix + x` | ペインを閉じる |
| `prefix + z` | ペインをズーム/解除 |
| `prefix + q` | ペイン番号表示 |

---

## コピーモード（vi キー）

| キー | 説明 |
|------|------|
| `prefix + [` | コピーモード開始 |
| `prefix + ]` | ペースト |
| `v` | 選択開始 |
| `y` | コピー |
| `q` | コピーモード終了 |
| `hjkl` | カーソル移動 |
| `/` | 前方検索 |
| `?` | 後方検索 |

---

## その他

| キー | 説明 |
|------|------|
| `prefix + r` | 設定ファイル再読込 |
| `prefix + t` | 時計表示 |
| マウス操作 | 有効（スクロール・ペイン選択） |
