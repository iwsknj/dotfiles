# Neovim チートシート（LazyVim ベース）

**Leader キー**: `Space`

---

## モード

| キー | 説明 |
|------|------|
| `i` | インサートモード |
| `jj` | インサートモード → ノーマルモード（カスタム） |
| `v` | ビジュアルモード |
| `V` | ビジュアルライン |
| `Ctrl-v` | ビジュアルブロック |
| `Esc` | ノーマルモードに戻る |

---

## カーソル移動

| キー | 説明 |
|------|------|
| `h/j/k/l` | 左/下/上/右（10回連続で "Hold it Cowboy!" 警告） |
| `j` / `k` | 折り返し行を考慮した移動（`gj`/`gk` にリマップ済み） |
| `w` / `b` | 単語単位で前後移動 |
| `0` / `$` | 行頭 / 行末 |
| `gg` / `G` | ファイル先頭 / 末尾 |
| `Ctrl-d` / `Ctrl-u` | 半ページ下/上 |

---

## hop.nvim（高速ジャンプ）

| キー | 説明 |
|------|------|
| `f{char}` | カーソル後の文字にジャンプ（現在行） |
| `F{char}` | カーソル前の文字にジャンプ（現在行） |
| `t{char}` | カーソル後の文字の1つ前にジャンプ（現在行） |
| `T{char}` | カーソル前の文字の1つ後にジャンプ（現在行） |
| `<leader><leader>j` | 下方向の単語にジャンプ |
| `<leader><leader>k` | 上方向の単語にジャンプ |
| `<leader><leader>hl` | 行ジャンプ（HopLine） |

---

## nvim-surround（囲み文字操作）

| キー | 説明 |
|------|------|
| `ys{motion}{char}` | モーション範囲を囲む（例: `ysiw"` で単語を `"` で囲む） |
| `ds{char}` | 囲み文字を削除（例: `ds"` で `"` を削除） |
| `cs{old}{new}` | 囲み文字を変更（例: `cs"'` で `"` → `'`） |
| `S{char}` | ビジュアル選択を囲む |

---

## vim-commentary（コメントアウト）

| キー | 説明 |
|------|------|
| `gcc` | 現在行をコメントアウト/解除 |
| `gc{motion}` | モーション範囲をコメントアウト |
| `gc` | ビジュアル選択をコメントアウト |

---

## ファイル操作・検索（LazyVim デフォルト）

| キー | 説明 |
|------|------|
| `<leader>ff` | ファイル検索（fzf） |
| `<leader>fg` | Grep 検索 |
| `<leader>fb` | バッファ一覧 |
| `<leader>e` | Neo-tree ファイルツリー開閉 |
| `<leader>E` | Neo-tree（現在ファイルの場所を開く） |

---

## Neo-tree（ファイルツリー）

| キー | 説明 |
|------|------|
| `<leader>e` | トグル |
| `a` | ファイル/ディレクトリ作成 |
| `d` | 削除 |
| `r` | リネーム |
| `y` | コピー |
| `x` | カット |
| `p` | ペースト |
| `h` / `l` | 親ディレクトリ / 子ディレクトリへ |
| `Enter` | ファイルを開く |

---

## バッファ・ウィンドウ

| キー | 説明 |
|------|------|
| `<leader>bd` | バッファを閉じる |
| `[b` / `]b` | 前/次のバッファ |
| `<leader>-` | ウィンドウを横分割 |
| `<leader>\|` | ウィンドウを縦分割 |
| `Ctrl-h/j/k/l` | ウィンドウ間移動 |
| `<leader>wq` | ウィンドウを閉じる |

---

## LSP

| キー | 説明 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバーでドキュメント表示 |
| `<leader>ca` | コードアクション |
| `<leader>cr` | リネーム |
| `<leader>cf` | フォーマット |
| `]d` / `[d` | 次/前の診断へ |
| `<leader>cd` | 診断の詳細表示 |

---

## 編集

| キー | 説明 |
|------|------|
| `+` | 無名レジスタの内容をシステムクリップボード（`+`レジスタ）にコピー（カスタム） |
| `u` / `Ctrl-r` | アンドゥ / リドゥ |
| `dd` / `yy` / `p` | 行削除 / 行コピー / ペースト |
| `ciw` / `diw` | 単語を変更 / 削除 |
| `>` / `<` | インデント追加 / 削除（ビジュアルモード） |

---

## その他

| キー | 説明 |
|------|------|
| `:w` | 保存 |
| `:q` | 終了 |
| `:wq` | 保存して終了 |
| `<leader>l` | Lazy（プラグイン管理）を開く |
| `<leader>cm` | Mason（LSP管理）を開く |
